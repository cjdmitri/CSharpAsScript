# CSharpAsScript
Пример компиляции и выполнения кода C# во время выполнения программы.
## Описание
Консольное приложение, которое при запуске читает файл класса `Script1.cs`, находящийся как отдельный текстовый файл в каталоге приложения.
- Далее происходит чтение и парсинг синтаксического дерева.
- Добавление неоходимых зависимостей
- Компиляция файла, как библиотеки классов
- Создание нового контекста для библиотеки
- Загрузка библиотеки в контекст
- Выполнение скрипта
- Выгрузка контекста
- Освобождение памяти.

## Дополнительно
Для демонстрации возможностей, в скрипт добавлена ссылка на консольное приложение и выполнение метода из данного приложения. А так же отслеживаются метрики: выделение памяти и время компиляции и выполнения.
